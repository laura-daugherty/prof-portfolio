Countries Compendium

Countries Compendium is a Single Page App that uses the REST countries API to display data about different countries. I used React, Typescript and Axios to execute the front end of this app. You can checkout my repo on Github

Shortly

Shortly is a simple one page application using the re.link API (discontinued after my project was finished) to shorten links. I used React and vanilla HTML and CSS to create this page. Check out my Github to learn more.

IP-Tracker

IP-Tracker is an app that looks up information about your IP address using the IP Geolocation API by IPify and displays the location of your ISP using LeafletJS and Mapbox. I built the front end of this app using React and Typescript and connected to the APIs using Axios. Check out my repo on Git

Tarot Flip

Tarot Flip is a Single Page App that gives the user a simple three card tarot reading. I designed and executed the front and backends using Hashrouter and semantic UI on the frontend. Tarot Flip uses a Node.js backend and Postgres DB hosted with Heroku to serve tarot cards to the frontend. FE Github BE Github

PlanFul

PlanFul is a simple portfolio single page application for wedding planners. I planned the front end state management using React Hooks and guided a team of other engineers to execute the app. The backend is built with Node.js and connects to the front end using Axios. Styling is done using Semantic UI. Check out my repo on Github.

I'll Serve Soup

I'll Serve Soup is a kitchen inventory single page application. It has CRUD functionality, a node backend, and a redux state management structure. I designed and executed the styling and implemented state throughout the App. You can see my repo on Github
